# 114 Songs

This is an attempt to digitize all of Charles Ives's 114 songs. View the current ones at [MuseScore.](https://musescore.com/user/95184583/sets/13552957)

The songs are all in the public domain and the original volume is avaliable for viewing at [Harvard.](https://iiif.lib.harvard.edu/manifests/view/drs:7784482$1i)
